Copyright (C) 2013, Petros Kyladitis


Description
===========
A simple utility for setup or change the OEM Infromation at MS Windows Vista and above.
You can set or change the entries for:
- Manufacturer
- Model
- Support Hours
- Support Tel
- Support URL
- Path to Logo

For updates and more info see at <http://multipetros.gr/public-projects/oemedit/>


Requirements
============
This program is designed for MS Windows Vista and above x86 or x64 architectures.
Also you need the .NET Framework 3.5 be installed on your system.
